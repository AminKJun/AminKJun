# Amin K. — Portfolio

This repository contains a simple, static portfolio website.

How to preview locally:
1. Open `index.html` in your browser.
2. Or run a simple server: `python -m http.server` then open http://localhost:8000

Deploy to GitHub Pages:
- Push these files to the default branch (usually `main`). In repo settings → Pages, choose the branch (root) as the source and save. The site will be available at `https://AminKJun.github.io/AminKJun` after a few minutes.

Replace placeholders:
- Update `index.html` sections with your real project links, descriptions, and add `resume.pdf` to the repo root.
- Replace images by creating an `images/` folder and linking them in the markup.

Contact form:
- The current form posts to Formspree; replace `https://formspree.io/f/your-form-id` with your Formspree endpoint or use a mailto link.

Enjoy!
